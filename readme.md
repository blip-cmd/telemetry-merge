# Telemetry Merge

## Overview
This repository documents a pair programming challenge completed by Ryan and Sakibu as part of the **Deloitte Australia Technology Virtual Internship**. We tackled two main tasks within a 2-hour timeframe.

## Tasks Completed

### Task 1: Coding
Developed a Python algorithm to merge two distinct data models into a unified format, enabling seamless data integration and standardization across multiple sources.

### Task 2: Development
Created a comprehensive software development proposal outlining the architecture and implementation plan for building a data dashboard to visualize merged telemetry data.

- **Duration**: 2 hours
